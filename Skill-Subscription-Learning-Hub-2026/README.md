# Skill Subscription Hub Project

## Overview

You are building a **Spring Boot + JSP full-stack application** where users can:
- Register and login
- View skill packs
- Subscribe to skill packs
- View their subscriptions

---

##  Architecture (DO NOT CHANGE)

Browser -> Controller -> Service -> Repository -> Database -> JSP

---

##  Tech Stack

- Java 17 or latest version
- Spring Boot
- Spring MVC
- Spring Data JPA
- JSP + JSTL
- MySQL (or H2)

---

##  Features

### User Module
- Register
- Login

### Skill Pack Module
- Add skill packs
- View all packs
- Update / Delete packs

### Subscription Module
- Subscribe user to pack
- View user subscriptions

---

##  Project Rules

- Do NOT change package structure
- Do NOT modify entity relationships
- Do NOT write SQL manually
- Service layer contains business logic
- Controller only handles routing

---

##  Final Flow

1. Register User
2. Login
3. View Skill Packs
4. Subscribe
5. View Subscriptions

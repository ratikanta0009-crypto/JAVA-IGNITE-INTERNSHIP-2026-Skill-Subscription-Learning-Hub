# Skill Subscription Hub — Project Overview

## What is this project?

This is a **full-stack web application** built using Spring Boot and JSP where users can:

- Create an account
- Login securely
- Browse available skill learning packs
- Subscribe to a skill pack
- View their active subscriptions

Think of it like a **mini learning platform (like Udemy website)** but simplified for training purposes.

---

##  Why are we building this?

This project is designed to help you understand:

✔ How real web applications are structured  
✔ How frontend (JSP) connects to backend (Spring Boot)  
✔ How business logic is separated from UI  
✔ How database operations are handled cleanly  

---

##  Real-world analogy

Imagine this system like a real company:

-  User → Student in learning platform  
-  Skill Pack -> Course (Java, Python, Spring Boot etc.)  
-  Subscription -> Enrollment in a course  

---

##  How the system works (FLOW)
User opens website
Registers account
Logs in
Views available skill packs
Subscribes to a pack
System stores subscription in database
User views all subscriptions

---

##  Architecture Understanding

This project follows MVC pattern:
Browser (JSP Pages)
|
Controller (Request Handler)
|
Service (Business Logic)
|
Repository (Database Access)
|
Database


---

##  Modules in this project

### 1. User Module
- Register user
- Login authentication

### 2. Skill Pack Module
- Add new skill packs
- View all packs

### 3. Subscription Module
- Subscribe user to a pack
- View user subscriptions
